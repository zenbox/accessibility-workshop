Tastatur-Steuerung

Tastatursteuerungen sind gemäß den Web Content Accessibility Guidelines (WCAG) insbesondere hilfreich für:

  • Menschen mit motorischen Beeinträchtigungen: Diese Nutzer haben möglicherweise Schwierigkeiten, eine Maus präzise zu bewegen, und sind daher auf die Tastatur angewiesen, um auf Webseiten zu navigieren.
  • Sehbehinderte oder blinde Menschen: Diese Nutzer verwenden häufig Screenreader und andere assistive Technologien, die hauptsächlich über die Tastatur gesteuert werden. Eine vollständig per Tastatur steuerbare Webseite ermöglicht es ihnen, Inhalte zu erfassen und zu navigieren. Screenreader bringen selbst eine Tastatursteuerung mit, diese darf durch die Webseite nicht beeinträchtigt werden.
  • Menschen ohne Zugang zu einer Maus: Dies schließt Nutzer ein, die temporär oder situativ keine Maus verwenden können, wie etwa bei bestimmten Geräten oder in bestimmten Umgebungen.

Breadcrumbs - Navigation

Eine Breadcrumbs-Navigation, die aus Links aufgebaut ist, benötigt keine zusätzliche Tastatursteuerung. Tab und Enter sind browserseitig vorhanden.

Im Grunde ist das eine Brotkrümel-Navigation. Das bedeutet, dass du dich in einer Navigation befindest, die dir den Weg durch die Seite zeigt. Du kannst mit der Tab-Taste durch die Links navigieren und mit der Enter-Taste die Links öffnen.

Telefonnummer

Telefonnummern können als Link mit hinterlegter Nummer gebaut werden. Eine zusätzliche Tastatursteuerung ist nicht notwendig. Ggf. ist ein erklärender Text hilfreich, der die Funktion "Anrufen" der Enter-Taste beschreibt.

Telefonnummer: +49 221 12 34 57 89-0

Die Telefonnummer kann über den Link angerufen werden. Dazu musst du den Link auswählen und anklicken. bzw. mit der enter-Taste bestätigen.

Die Tastatursteuerung

Right Arrow

Verschiebt den Fokus eine Zelle nach rechts.

Verschiebt den Fokus eine Zelle nach rechts.

Wenn der Fokus auf der äußersten rechten Zelle in der Zeile liegt, verschiebt sich der Fokus nicht.

Verschiebt den Fokus eine Zelle nach rechts.

Left Arrow

Verschiebt den Fokus eine Zelle nach links.

Wenn der Fokus auf der äußersten linken Zelle in der Zeile liegt, verschiebt sich der Fokus nicht.

Down Arrow

Verschiebt den Fokus eine Zelle nach unten.

Wenn der Fokus auf der untersten Zelle in der Spalte liegt, wird der Fokus nicht verschoben.

Up Arrow

Verschiebt den Fokus eine Zelle nach oben.

Wenn der Fokus auf der obersten Zelle in der Spalte liegt, verschiebt sich der Fokus nicht.

Page Down

In Beispiel 3 wird der Fokus fünf Zeilen nach unten verschoben, wobei ein Bildlauf durchgeführt wird, sodass die unterste Zeile im aktuell sichtbaren Zeilensatz zur ersten sichtbaren Zeile wird.

Page Up

In Beispiel 3 wird der Fokus 5 Zeilen nach oben verschoben, wobei so gescrollt wird, dass die oberste Zeile im aktuell sichtbaren Zeilensatz die letzte sichtbare Zeile wird.

Wenn der Fokus in der ersten Zeile des Rasters liegt, verschiebt sich der Fokus nicht.

Die Tastatursteuerung der Menüleiste

Space Enter

Wenn es sich bei dem Element um ein übergeordnetes Menüelement handelt, wird das Untermenü geöffnet und der Fokus auf das erste Element im Untermenü verschoben.

Andernfalls wird das Menüelement aktiviert, das neuen Inhalt lädt und den Fokus auf die Überschrift setzt, die den Inhalt betitelt.

Right Arrow

Verschiebt den Fokus auf das nächste Element in der Menüleiste.

Wenn der Fokus auf dem letzten Element liegt, wird der Fokus auf das erste Element verschoben.

Left Arrow

Verschiebt den Fokus auf das vorherige Element in der Menüleiste.

Wenn der Fokus auf dem ersten Element liegt, wird der Fokus auf das letzte Element verschoben.

Down Arrow

Öffnet das Untermenü und verschiebt den Fokus auf das erste Element im Untermenü.

Up Arrow

Öffnet das Untermenü und verschiebt den Fokus zum letzten Element im Untermenü.

Home

Verschiebt den Fokus auf das erste Element in der Menüleiste.

End

Verschiebt den Fokus auf das letzte Element in der Menüleiste.

Character

Verschiebt den Fokus auf das nächste Element in der Menüleiste, dessen Name mit dem eingegebenen Zeichen beginnt.

Wenn der Name keines der Elemente mit dem eingegebenen Zeichen beginnt, wird der Fokus nicht verschoben.

Die Tastatursteuerung der Untermenüs

Space Enter

Wenn es sich bei dem Element um ein übergeordnetes Menüelement handelt, wird das Untermenü geöffnet und der Fokus auf das erste Element im Untermenü verschoben.

Andernfalls wird das Menüelement aktiviert, das neuen Inhalt lädt und den Fokus auf die Überschrift setzt, die den Inhalt betitelt.

Escape

Schließt das Untermenü.

Verschiebt den Fokus auf das übergeordnete Menüleistenelement.

Right Arrow

Wenn der Fokus auf einem Element mit einem Untermenü liegt, wird das Untermenü geöffnet und der Fokus auf das erste Element gesetzt.

Wenn der Fokus auf einem Element liegt, das kein Untermenü hat:

  • Schließt das Untermenü.
  • Verschiebt den Fokus zum nächsten Element in der Menüleiste.
  • Öffnet das Untermenü des neu fokussierten Menüleistenelements und behält den Fokus auf dem übergeordneten Menüleistenelement.
Left Arrow

Schließt das Untermenü und verschiebt den Fokus zum übergeordneten Menüelement.

Wenn sich das übergeordnete Menüelement in der Menüleiste befindet, gilt außerdem:

  • verschiebt den Fokus zum vorherigen Element in der Menüleiste.
  • Öffnet das Untermenü des neu fokussierten Menüleistenelements und behält den Fokus auf dem übergeordneten Menüleistenelement.
Down Arrow

Verschiebt den Fokus auf das nächste Element im Untermenü.

Wenn der Fokus auf dem letzten Element liegt, wird der Fokus auf das erste Element verschoben.

Up Arrow

Verschiebt den Fokus zum vorherigen Element im Untermenü.

Wenn der Fokus auf dem ersten Element liegt, wird der Fokus auf das letzte Element verschoben.

Home

Verschiebt den Fokus auf das erste Element im Untermenü.

End

Verschiebt den Fokus auf das letzte Element im Untermenü.

Character

erschiebt den Fokus auf das nächste Element, dessen Name mit dem eingegebenen Zeichen beginnt.

Wenn der Name keines der Elemente mit dem eingegebenen Zeichen beginnt, wird der Fokus nicht verschoben.
