# **Tabellarische Übersicht: Vergleich der Formulierungen + angewendete Regeln**

| Sprachstufe                           | Formulierung                                                         | Erläuterung: Welche Regeln wurden angewendet?                                                                                     |
| ------------------------------------- | -------------------------------------------------------------------- | --------------------------------------------------------------------------------------------------------------------------------- |
| **Originalsatz**                      | „Bitte überprüfen Sie Ihre Angaben, bevor Sie den Kauf abschließen.“ | Nicht vereinfacht; komplexer Satzbau; übliche E-Commerce-Sprache.                                                                 |
| **Hamburger Verständlichkeitsmodell** | „Bitte prüfen Sie Ihre Angaben, bevor Sie kaufen.“                   | • Komplexität reduziert<br>• Satz verkürzt<br>• Fachwort „abschließen“ → alltagssprachlich „kaufen“<br>• Struktur bleibt erhalten |
| **Einfache Sprache**                  | „Bitte prüfen Sie Ihre Angaben. Danach können Sie kaufen.“           | • Ein Nebensatz wurde aufgeteilt<br>• Zwei kurze Sätze statt einem langen<br>• Einfachere Syntax<br>• Verständlicheres Verb       |
| **Leichte Sprache**                   | „Bitte prüfen Sie Ihre Angaben. Danach kaufen Sie.“                  | • Sehr kurze Sätze<br>• Keine Nebensätze<br>• Direkte Ansprache<br>• Max. 6–8 Wörter/Satz<br>• Aktive Formulierung                |
| **Begründung**                        | —                                                                    | Je stärker die Vereinfachung, desto mehr Strukturierung und Satzteilung ist erforderlich.                                         |

---

### **Zweites Beispiel (Produktbeschreibung)**

| Sprachstufe          | Formulierung                                         | Erläuterung                                                                               |
| -------------------- | ---------------------------------------------------- | ----------------------------------------------------------------------------------------- |
| **Original**         | „Dieser Artikel ist derzeit nur begrenzt verfügbar.“ | Unklare Mengenangabe; abstrakte Formulierung.                                             |
| **Hamburger Modell** | „Von diesem Artikel gibt es nur wenige Stück.“       | • Abstrakte Formulierung konkretisiert<br>• Einfachere Wortwahl                           |
| **Einfache Sprache** | „Es gibt nur wenige Stück von diesem Artikel.“       | • Alltagswortschatz<br>• Klare Reihenfolge Subjekt → Aussage                              |
| **Leichte Sprache**  | „Es gibt nur wenige Stück.“                          | • Kürzerer Satz<br>• Fokus auf Kerninformation<br>• Wiederholung, falls nötig, erlaubt    |
| **Begründung**       | —                                                    | Leichte Sprache maximiert Verständlichkeit und Lesbarkeit für sehr geringe Lesekompetenz. |

---

### **Drittes Beispiel (Lieferzeit)**

| Sprachstufe          | Formulierung                                                               | Regeln / Erläuterung                                                              |
| -------------------- | -------------------------------------------------------------------------- | --------------------------------------------------------------------------------- |
| **Original**         | „Die Lieferzeit beträgt 2–4 Werktage.“                                     | Kompakt, aber Fachbegriff „Werktage“                                              |
| **Hamburger Modell** | „Die Lieferung dauert 2 bis 4 Werktage.“                                   | • aktivere Formulierung<br>• Zahlen ausgeschrieben                                |
| **Einfache Sprache** | „Die Lieferung dauert 2 bis 4 Tage. Das sind Arbeits-Tage.“                | • Fachwort erklärt<br>• Zusatz zur Orientierung                                   |
| **Leichte Sprache**  | „Die Lieferung dauert 2 bis 4 Tage.“<br>„Das heißt: 2 bis 4 Arbeits-Tage.“ | • Zwei sehr kurze Sätze<br>• Erklärung getrennt<br>• leichter Wortschatz          |
| **Begründung**       | —                                                                          | Leichte Sprache teilt immer Informationen, die im Original kompakt gesagt werden. |

---

Lösung für ### **Fehlermeldung (Originalversion)**

„Beim Verarbeiten Ihrer Zahlung ist ein Fehler aufgetreten. Bitte überprüfen Sie Ihre Zahlungsinformationen und versuchen Sie es erneut. Wenn das Problem weiterhin besteht, kontaktieren Sie unseren Kundensupport für weitere Hilfe.“

### Lösung nach dem **Hamburger Verständlichkeitsmodell**

„Bei der Verarbeitung Ihrer Zahlung ist ein Fehler passiert. Bitte prüfen Sie Ihre Zahlungsinformationen und versuchen Sie es noch einmal. Wenn das Problem weiter besteht, wenden Sie sich an unseren Kundensupport. Er hilft Ihnen.“

Tabelle der angewendeten Regeln:
| Regel | Erläuterung |
| --------------------------------------- | ------------------------------------------------------------------------------------------------ |
| Komplexität reduzieren | „Verarbeiten“ → „Verarbeitung“; „aufgetreten“ → „passiert“ |
| Satzstruktur beibehalten | Drei Sätze bleiben erhalten |
| Fachbegriffe erklären | „Kundensupport“ bleibt, da allgemein verständlich |
| Aktiv statt Passiv | „kontaktieren Sie unseren Kundensupport“ → „wenden Sie sich an unseren Kundensupport“ |

### Lösung in **Einfacher Sprache**

„Bei der Verarbeitung Ihrer Zahlung ist ein Fehler passiert. Bitte prüfen Sie Ihre Zahlungsdaten. Versuchen Sie es noch einmal. Wenn das Problem weiter besteht, rufen Sie den Kundensupport an. Er hilft Ihnen.“

Tabelle der angewendeten Regeln:
| Regel | Erläuterung |
| --------------------------------------- | ------------------------------------------------------------------------------------------------ |
| Lange Sätze aufteilen | Drei Sätze in vier Sätze aufgeteilt |
| Einfache, alltägliche Wörter verwenden | „Zahlungsinformationen“ → „Zahlungsdaten“; „kontaktieren“ → „rufen ... an“ |
| Klare Reihenfolge Subjekt → Aussage | „versuchen Sie es erneut“ → „versuchen Sie es noch einmal“ |

### Lösung in **Leichter Sprache**

Bei der Verarbeitung Ihrer Zahlung ist ein Fehler passiert.

Bitte prüfen Sie Ihre Zahlungs-Daten.

Zum Beispiel Ihre Kontonummer.

Oder Ihre Kredit-Karten-Nummer.

Dann versuchen Sie es noch einmal.

Ist der Fehler noch da?

Dann rufen Sie den Kunden-Support an.

Er hilft Ihnen.

Kunden-Support heißt: Sie können dort anrufen oder eine E-Mail schreiben. Dort arbeiten Menschen, die Ihnen helfen.

Tabelle der angewendeten Regeln:
| Regel | Erläuterung |
| --------------------------------------- | ------------------------------------------------------------------------------------------------ |
| Sehr kurze Sätze | Jeder Satz hat max. 6–8 Wörter |
| Keine Nebensätze | Alle Sätze sind Hauptsätze |
| Konkrete Wörter | „Zahlungsinformationen“ → „Zahlungsdaten“ |
| Aktiv statt Passiv | „kontaktieren Sie unseren Kundensupport“ → „rufen Sie den Kundensupport an“ |
